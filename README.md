## ğŸš€ Getting Started

### Prerequisites

- Node.js and npm
- Sass compiler

### Installation

1. Clone the repository:

```bash
git clone https://github.com/zbaharyilmaz/sass-dune-movie-website.git
cd sass-dune-movie-website
```

2. Install dependencies:

```bash
npm install
```

3. Compile Sass:

```bash
npm run build-css
```

4. Watch for changes:

```bash
npm run watch-css
```

5. Start local server:

```bash
npm run serve
```
